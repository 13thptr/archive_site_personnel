Les types de gladiateurs

Les gladiateurs appartenaient à différentes catégories appelées armaturae qui définissaient leur équipement,techniques de combat, ceci afin de représenter les différents peuples auxquels les gladiateurs appartenaient et rendre les combats imprévisibles.

Parmi les types principaux, on peut citer:
  • Le Mirmillon
  • Le Rétiaire
  • Le Thrace
  • Le Secutor
  • Le Bestiaire
